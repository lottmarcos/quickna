# App

Quick & A App Project!

## requirements

To run this project, you will need the following:

- Node.js >= 20.0.0
- npm >= 8.0.0
- Yarn >= 1.22.22
- Docker
- Docker Compose

## environment variables:

- POSTGRES_HOST
- POSTGRES_PORT
- POSTGRES_USER
- POSTGRES_DB
- POSTGRES_PASSWORD
- POSTGRES_SSLMODE
- POSTGRES_POOL_MODE
- DATABASE_URL
